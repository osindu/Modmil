HOSTING on HEROKU

What is Heroku?

Heroku is a free hosting site that can host many web apps. However, the web apps cannot store any data. We have made Mod Mail to do exactly that. It was made to be stateless and not store any data in json files or any other storage files.

How do I do it?

If you choose to install the bot using Heroku, you do not need to download anything. In fact, you can set it all up on a phone!

COMMANDS

Parameters are either [] or <>.

[] - Optional Parameters

<> - Mandatory Parameters

Do not include these while typing your command

Include your prefix in front of every commmand

`m.setup [modrole] <- (optional)` - Command that sets up the bot.
`m.reply ` - Sends a message to the current thread\'s recipient.
`m.close` - Closes the current thread and deletes the channel.
`m.disable` - Closes all threads and disables modmail for the server.
`m.customstatus` - Sets the Bot status to whatever you want. Resets daily!
`m.say ` — Writes as ModMail anything you send after command.
`m.ping` — Check you websocket latency (is ModMail alive?).
